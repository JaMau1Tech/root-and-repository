# Bash Notes

## What is Bash?
Bash (Bourne Again Shell) is the default command-line shell on many Linux distributions.

## Basic Navigation Commands

```bash
pwd
```
Displays the current directory.

```bash
ls
```
Lists files and directories.

```bash
ls -la
```
Shows hidden files and detailed information.

```bash
cd directory-name
```
Changes directory.

```bash
cd ..
```
Moves up one directory.

## File Commands

```bash
mkdir foldername
```
Creates a new directory.

```bash
touch filename.txt
```
Creates a new file.

```bash
cp source destination
```
Copies a file.

```bash
mv source destination
```
Moves or renames a file.

```bash
rm filename
```
Deletes a file.

```bash
rm -r foldername
```
Deletes a directory.

## Helpful Commands

```bash
clear
```
Clears the terminal.

```bash
history
```
Displays previously used commands.

```bash
man command
```
Shows command documentation.
