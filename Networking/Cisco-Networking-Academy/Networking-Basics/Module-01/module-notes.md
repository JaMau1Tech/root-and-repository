## Module Overview

This module introduced the fundamental concepts of computer networking. It explained what the Internet is, how local networks communicate, how data is represented and transmitted, and how network performance is measured. These concepts form the foundation for all future networking topics.

---

## Learning Objectives

- Explain what the Internet is and how it operates.
- Differentiate between local networks and the Internet.
- Identify different network types.
- Understand how computers represent and transmit data.
- Explain binary, bits, and bytes.
- Identify common methods of data transmission.
- Differentiate between bandwidth and throughput.
- Understand latency and bottlenecks.

---

## Topics Covered

### 1.1 Network Types

Topics

- Internet
- Local Area Networks (LAN)
- SOHO Networks
- Business Networks
- Internet of Things (IoT)

Key Concepts

- The Internet is a worldwide collection of interconnected networks.
- No single organization owns the Internet.
- Local networks connect nearby devices.
- Businesses use larger networks to share resources and improve communication.
- Modern devices such as smart TVs, smartwatches, vehicles, and appliances are connected through IoT.

---

### 1.2 Data Transmission

Topics

- Personal data
- Binary
- Bits and bytes
- ASCII
- Signal transmission

Key Concepts

- Computers only understand binary (0 and 1).
- A bit is the smallest unit of digital data.
- Eight bits equal one byte.
- Data is converted into signals before being transmitted across a network.
- Networks use electrical, optical, and wireless signals to transfer information.

---

### 1.3 Bandwidth and Throughput

Topics

- Bandwidth
- Throughput
- Latency
- Bottlenecks
- Speed testing

Key Concepts

- Bandwidth represents the maximum capacity of a network connection.
- Throughput represents the actual amount of data successfully transferred.
- Latency measures communication delay.
- The slowest link in a network path creates a bottleneck.
- Speed tests estimate real-world network throughput.
