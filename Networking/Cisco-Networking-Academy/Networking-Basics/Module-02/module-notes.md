# Topics Covered

## 2.1 Clients and Servers

### Client-Server Model

- Hosts can function as clients, servers, or both.
- Client software requests resources.
- Server software provides resources.
- A computer's role depends on the software it is running.

### Common Server Types

- Web Server
- Email Server
- File Server

### Peer-to-Peer Networks

- Computers share resources directly.
- No dedicated server required.
- Simple and inexpensive.
- Limited security and scalability.

### Peer-to-Peer Applications

- Devices function as both client and server.
- Resource sharing occurs directly between peers.

### Multiple Network Roles

- A server can provide multiple services simultaneously.
- A client can communicate with multiple servers at the same time.

### Intermediate Devices

## 2.3 ISP Connectivity

### Internet Service Providers

An ISP connects a home or business network to the Internet.

Common ISP responsibilities include:

- Internet access
- Technical support
- Hosting services
- Voice over IP services

### Internet Backbone

- Global high-speed fiber-optic infrastructure
- Connects ISPs worldwide
- Enables communication between networks

### Home Network Connectivity

Typical communication path:

```text
End Device
    ↓
Router
    ↓
Modem
    ↓
ISP
    ↓
Internet
    ↓
Remote Server
```

### Internet Connection Types

#### Cable

- Uses coaxial cable
- Common residential solution
- High-speed broadband

#### DSL

- Uses telephone lines
- Speed depends on distance from provider

#### Cellular

- Mobile broadband
- Portable Internet access

#### Satellite

- Useful for rural locations
- Higher latency than wired connections

#### Dial-Up

- Telephone modem
- Very slow
- Legacy technology
