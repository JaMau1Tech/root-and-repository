# OSI Model

## Layer 7 – Application
Provides services to applications.

Examples:
- HTTP
- HTTPS
- FTP
- DNS

## Layer 6 – Presentation
Handles data formatting and encryption.

Examples:
- SSL
- TLS
- JPEG
- PNG

## Layer 5 – Session
Creates and manages sessions between devices.

Examples:
- RPC
- NetBIOS

## Layer 4 – Transport
Provides reliable communication.

Protocols:
- TCP
- UDP

## Layer 3 – Network
Responsible for routing packets.

Protocols:
- IP
- ICMP

Devices:
- Routers

## Layer 2 – Data Link
Responsible for MAC addressing.

Devices:
- Switches

## Layer 1 – Physical
Transfers raw bits.

Examples:
- Ethernet cables
- Fiber optics
