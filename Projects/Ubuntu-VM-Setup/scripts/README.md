# Scripts

This folder is reserved for scripts related to the Ubuntu VM Setup project.

No scripts were required during the initial setup because the installation and troubleshooting process was completed manually.

---

# Purpose

This folder will store future automation scripts for:

- Updating the system
- Installing common tools
- Collecting system information
- Creating backups
- Checking network settings
- Performing routine maintenance
