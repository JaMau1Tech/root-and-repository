# Key Takeaways

- The OSI Model contains seven layers.
- Each layer has a distinct responsibility.
- TCP provides reliable communication, while UDP prioritizes speed.
- Routing occurs at Layer 3.
- MAC addresses operate at Layer 2.
- Encryption is handled at Layer 6.
- Applications communicate through Layer 7.
- Understanding the OSI Model is fundamental for networking and cybersecurity.
