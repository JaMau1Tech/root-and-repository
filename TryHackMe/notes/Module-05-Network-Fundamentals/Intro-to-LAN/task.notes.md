# Key Takeaways

- LANs connect devices within a limited area.
- Different network topologies have unique strengths and weaknesses.
- Switches efficiently forward traffic within a LAN.
- Routers connect separate networks through routing.
- Subnetting improves organization, efficiency, and security.
- ARP maps logical IP addresses to physical MAC addresses.
- DHCP automates IP address assignment using the DORA process.

# Lessons Learned

- Different network topologies provide different trade-offs between cost, scalability, and reliability.
- Subnetting improves network organization and security.
- ARP enables communication between devices by mapping IP addresses to MAC addresses.
- DHCP greatly simplifies network administration through automatic IP address assignment.
- These networking concepts provide the foundation for understanding enterprise networks and cybersecurity.
