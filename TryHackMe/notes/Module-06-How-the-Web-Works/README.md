# Module 06 – How The Web Works

## Overview

This module introduces the core technologies that make the World Wide Web function. It covers how domain names are resolved, how browsers communicate with web servers, how websites are built, and how all of these technologies work together.

These concepts form the foundation for later topics in web application security, penetration testing, and network analysis.

---

## Module Objectives

- Understand the purpose of DNS
- Learn how HTTP works
- Understand how websites are built
- See how web technologies work together
- Build a strong foundation for web security
