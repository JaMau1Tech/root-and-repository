## Overview

This room introduced the Domain Name System (DNS), explaining how it translates human-readable domain names into IP addresses. It covered the DNS hierarchy, common DNS record types, the DNS resolution process, and practical DNS queries.

---

## Objectives

- Understand the purpose of DNS
- Learn the DNS hierarchy
- Identify common DNS record types
- Understand DNS resolution
- Practice querying DNS records

---

## Key Concepts

### Domain Name System (DNS)

- Converts domain names into IP addresses
- Simplifies internet navigation

### DNS Hierarchy

- Root Domain
- Top-Level Domain (TLD)
- Second-Level Domain (SLD)
- Subdomain

### Record Types

| Record | Purpose |
|---------|----------|
| A | IPv4 Address |
| AAAA | IPv6 Address |
| CNAME | Alias |
| MX | Mail Server |
| TXT | Text / SPF / DKIM / DMARC |

### DNS Resolution

1. Local Cache
2. Recursive Resolver
3. Root Server
4. TLD Server
5. Authoritative Server
6. Response Cached (TTL)

### Practical Skills

- Query A records
- Query CNAME records
- Query MX records
- Query TXT records
