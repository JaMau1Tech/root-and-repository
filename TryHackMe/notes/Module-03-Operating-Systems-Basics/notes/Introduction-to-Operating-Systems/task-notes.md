## Key Concepts Learned

### Operating System (OS)

Core software that manages hardware, software, and system resources.

---

### Kernel

The core component of an operating system responsible for communicating directly with hardware.

---

### Kernel Space

A privileged area where the operating system kernel executes with unrestricted hardware access.

---

### User Space

A protected environment where applications execute with limited permissions.

Applications communicate with hardware through system calls handled by the kernel.

---

### Graphical User Interface (GUI)

Allows users to interact with the operating system using windows, icons, menus, and graphical elements.

---

### Command-Line Interface (CLI)

Allows users to interact with the operating system by entering text-based commands.

The CLI offers greater speed, precision, and automation, making it an essential tool in cybersecurity.
