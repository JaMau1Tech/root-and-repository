## Lab / Hands-On

### Task 3 – Gathering System Information

1. Open Command Prompt.

2. Display the current logged-in user:

   ```cmd
   whoami
   ```

3. Display the computer name:

   ```cmd
   hostname
   ```

4. View Windows system information:

   ```cmd
   systeminfo
   ```

5. Record the following:

   - OS Name
   - OS Version
   - System Type

6. Display the network configuration:

   ```cmd
   ipconfig
   ```

7. Record the following:

   - IPv4 Address
   - Default Gateway
