# JavaScript: Simple Demo

## Room Information

- **Room:** JavaScript: Simple Demo
- **Module:** Module 04 – Software Basics
- **Difficulty:** Medium

---

## Objective

Learn the fundamentals of JavaScript programming by building a simple "Guess the Number" game using variables, constants, user input, conditional statements, and loops while running the program with Node.js.

---

## Topics Covered

- JavaScript Basics
- Variables
- Constants
- User Input
- Random Number Generation
- Conditional Statements
- While Loops
- Program Flow
- Node.js

---

## Key Concepts

### Variables

```javascript
let
```

Used for values that change during program execution.

Examples:

- `tries`
- `guess`

---

### Constants

```javascript
const
```

Used for values that remain unchanged.

Example:

- `secret`

---

### Random Numbers

```javascript
Math.random()
Math.floor()
```

Used together to generate a random integer between 1 and 20.

---

### Display Output

```javascript
console.log()
```

Displays information to the terminal.

---

### User Input

```javascript
rl.question()
```

Reads user input as text.

---

### Type Conversion

```javascript
parseInt(text, 10)
```

Converts text into an integer.

---

### Conditional Statements

```javascript
if
else if
else
```

Used to compare the user's guess against the secret number and provide feedback.

---

### Loops

```javascript
while
```

Repeats the guessing process until the correct number is entered.

## Lessons Learned

This room demonstrated that JavaScript follows the same programming logic as Python while using different syntax. Variables, conditionals, loops, and functions work together to create a complete interactive application, reinforcing core programming concepts across multiple languages.
