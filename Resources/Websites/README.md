# Websites

This folder tracks trusted websites, learning platforms, technical communities, and official online resources used throughout my IT and cybersecurity development.

## Purpose

- Maintain a curated list of trusted websites
- Organize learning platforms by topic
- Track useful documentation and lab environments
- Support continuous technical development
- Reduce reliance on random or unverified search results
